# owen-codes.com

Personal website for Owen Farthing — full-stack engineer and web application specialist.

## What's here

- **About** — who I am, skills, experience, and where to find me online
- **Playground** — interactive demos powered by live public APIs, showcasing React patterns like data tables, charts, pipelines, and media cards

## Built with

- React 19 + TypeScript + Vite
- Framer Motion for animations
- Recharts for data visualization
- TanStack Table for headless table logic
- CSS Modules + custom properties (no UI framework)

## Data sources

All data is fetched from free, public APIs — no keys required.

| Section | API |
|---------|-----|
| Countries table | [REST Countries](https://restcountries.com) |
| Weather KPIs + chart | [Open-Meteo](https://open-meteo.com) |
| Trending books pipeline | [Open Library](https://openlibrary.org) |
| Artwork media card | [Art Institute of Chicago](https://api.artic.edu) |

## Running locally

```bash
npm install
npm run dev
```
